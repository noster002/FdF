# FdF
42 School - Level02

This graphical project illustrates different maps as wireframe models of a 3d landscape using the miniLibX.

Check out [this guide](https://harm-smits.github.io/42docs/libs/minilibx) for everything about the miniLibX including the setup.

## Usage

`make` compiles the program.

As an argument it takes an fdf file (only with .fdf extension)
```
$ ./fdf <file.fdf>
```
containing only a rectangular map with integers.
